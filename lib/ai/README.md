# lib/ai

AI integration lives here (wired up from Phase 1 onward):

- `gemini.ts` — Gemini 2.5 Flash / Flash-Lite config (high-volume routes)
- `claude.ts` — Claude Sonnet config (reasoning routes: readiness, insights, panic)
- `prompts/` — version-controlled system prompts, one per feature

Phase 0 ships this directory empty by design — no AI code yet.
